# 学生成绩管理系统

一个简洁高效的学生成绩管理Web应用，使用纯HTML/CSS/JavaScript开发。

## 🚀 功能特点

- ✅ **学生信息管理** - 添加、编辑、删除学生信息
- ✅ **成绩自动计算** - 自动计算平均分和等级评定
- ✅ **智能搜索排序** - 按学号、姓名搜索，多条件排序
- ✅ **数据导出** - 导出为CSV格式文件
- ✅ **本地存储** - 使用localStorage持久化数据
- ✅ **响应式设计** - 适配电脑、平板、手机

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **存储**: 浏览器LocalStorage
- **样式**: CSS Grid, Flexbox, CSS变量
- **图标**: Font Awesome

## 📦 安装使用

### 方法一：直接使用
1. 下载 `index.html` 文件
2. 用浏览器打开即可使用

### 方法二：部署到服务器
1. 将文件上传到Web服务器
2. 通过网址访问使用

## 🎯 使用说明

### 添加学生
1. 点击"添加学生"按钮
2. 填写学号、姓名等基本信息
3. 输入各科成绩（可选）
4. 点击"保存"

### 管理成绩
- **编辑**: 点击行中的编辑按钮 ✏️
- **删除**: 点击行中的删除按钮 🗑️
- **搜索**: 在搜索框输入学号或姓名
- **排序**: 使用下拉菜单选择排序方式

### 数据导出
1. 点击"导出CSV"按钮
2. 浏览器会自动下载CSV文件
3. 可用Excel打开查看
